MCM-Controller UI (MCM6000\_UI)
=================
The MCM-Controller UI (legacy name MCM6000\_UI) is a developer tool
    for configuring the system and a production tool for installing the system.

This UI supports the MCM301, MCM6101, and MCM6201 as well as their respective FTDI ports.

Requirements
------------
- `make onewire` for the project "look-up-tables"
- DevOps feed [tho-img-pkgs](https://pkgs.dev.azure.com/Thor-VA-ENG/4635d239-c724-4dcb-bf30-17e5738ac7f5/_packaging/tho-img-pkgs/nuget/v3/index.json)
    - This is required to download the NuGet packages used by the project.
